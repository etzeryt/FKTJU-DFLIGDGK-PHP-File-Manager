#  PHP-File-Manager

> 这是一个使用 `php` 编写的文件管理器，您只需要上传一个文件，即可轻松地帮助您完成対网站目录文件的管理。

## 注意！！

 - 为保证安全，请在下载之后更改 `fm.php` 中的默认密码（13至15行处）。
 - 如果您不喜欢界面的背景图片，请修改 `fm.php` 中 1769 至 1771 行处的图片 URL。
 - 麻烦给个 ★ star 。

## 功能如下

 - （批量/单个）复制文件
 - （批量/单个）移动文件
 - （批量/单个）删除
 - （批量/单个）压缩/解压缩文件
 - 支持文件权限更改（Linux下）
 - 可设定密码登入（支持多用户）
 - 预览文件
 - 代码高亮（highlight.js）
 - 上传/下载文件
 - 新建文件夹
 - 重命名文件
 - 查看 `MIME 类型` （fileinfo 拓展）
 - ...

## 环境支持

 - `PHP` 版本大于等于 5.2 
 - `php_fileinfo`，`php_curl`，`php_iconv`，`php_mbstring`
 - Windows/Linux
 - IIS/Apache/Nginx

## 特性

 - 单个文件，轻量级
 - 无需数据库
 - 无需 `Rewrite`
 - 适应于绝大多数服务器环境
 - 上传即可使用
 - `bootstrap` 框架，响应式设计，适合于多种设备
